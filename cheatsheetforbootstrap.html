Cheat-Sheet for Bootstrap

How to link bootstrap in your Website

If you want to use Bootstrap you have to link it via this link in your header.

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

Text

Alignment

You can either align the text in the center, left, right justify. You use it through the classes which are shown in the code box below.

This text is aligned to the center.

This text is aligned to the left.

This text is aligned to the right.

This text is justified, meaning it is spaced out to fill the width of the container.

<p class="text-center">This text is aligned to the center.</p>
<p class="text-start">This text is aligned to the left.</p>
<p class="text-end">This text is aligned to the right.</p>
<p class="text-justify">This text is justified, meaning it is spaced out to fill the width of the container.</p>

  • Item 3

Inline Lists

Display list items horizontally using the list-inline class.

  • Item 1
  • Item 2
  • Item 3

<ul class="list-inline">
<li class="list-inline-item">Item 1</li>
<li class="list-inline-item">Item 2</li>

List Groups

Use the list-group class to create a list group, useful for displaying a list of content blocks. Each list item has is own action when clicked. In this case going to the top of the page.

Item 1 Item 2 Item 3

<div class="list-group">
<a href="#" class="list-group-item list-group-item-action">Item 1</a>
<a href="#" class="list-group-item list-group-item-action">Item 2</a>
<a href="#" class="list-group-item list-group-item-action">Item 3</a>
</div>

Buttons

You can create a basic button using the btn class.

<button type="button" class="btn btn-primary">Primary Button</button>

You can style these buttons using Contextual Button Styles. Apply contextual styles to buttons, such as btn-primary, btn-secondary, btn-success, btn-danger, btn-warning, btn-info, etc.

<button type="button" class="btn btn-success">Success Button</button>
<button type="button" class="btn btn-danger">Danger Button</button>

Adjust the size of buttons using btn-lg (large), btn-sm (small), or btn-xs (extra small).

<button type="button" class="btn btn-primary btn-lg">Large Button</button>
<button type="button" class="btn btn-secondary btn-sm">Small Button</button>

Tables

  • Basic Table:

    Creates a basic table with default styling.

    <table class="table">

  • Striped Rows:

    Alternates row background colors for better readability.

    <table class="table table-striped">

  • Bordered Table:

    Adds borders to the table and its cells.

    <table class="table table-bordered">

  • Hover Rows:

    Applies a hover effect to highlight rows when hovered.

    <table class="table table-hover">

  • Responsive Table:

    Ensures the table is horizontally scrollable on smaller screens.

    <div class="table-responsive">

  • Contextual Classes:

    Apply contextual classes to rows or cells for different styles.

    <tr class="table-primary">

  • Head Dark/Light:

    Sets the background color of the table header.

    <thead class="table-dark"> or <thead class="table-light">
