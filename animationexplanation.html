Animations

In this example, we're creating a CSS animation named wanderingMotion that simulates the movement of an element wandering around the screen. Let's break down how this animation works:

  1. The @keyframes rule defines the animation's keyframes, which are the stages of the animation's timeline. Each keyframe specifies how the element should appear at a certain point in time during the animation.

    • 0%: At the start of the animation (0% of the animation duration), the element is translated to its initial position (0, 0).
    • 12.5%, 25%, 37.5%: These keyframes define intermediate positions where the element is translated to different coordinates on the screen.
    • 50%: Halfway through the animation, the element returns to its original position (0, 0).
    • 62.5%, 75%, 87.5%: These keyframes define intermediate positions where the element is translated to different coordinates on the screen in the opposite direction.
    • 100%: At the end of the animation (100% of the animation duration), the element returns to its initial position (0, 0).

  2. The animation wanderingMotion is applied to the button with the class wandering-button.

    • .wandering-button: This class styles the button element and positions it absolutely at the center of the screen (top: 50%; left: 50%;). The button will be displayed at the center of the screen regardless of the viewport size.
    • animation: wanderingMotion 10s linear infinite;: This line specifies the animation to apply (wanderingMotion), sets the animation duration to 10s (10 seconds), specifies the timing function (linear, meaning the animation progresses at a constant speed), and sets the animation iteration count to infinite (the animation repeats indefinitely).

  3. The button is wrapped inside an anchor (<a>) tag with the href attribute pointing to animationexplanation.html. This makes the button clickable and redirects the user to the animationexplanation.html page when clicked.

HTML-Code

<a href="animationexplanation.html">
<button class="wandering-button">Click Me</button>
</a>

CSS-Code

@keyframes wanderingMotion {
0% {
transform: translate(0, 0);
}
12.5% {
transform: translate(10rem, 10rem);
}
25% {
transform: translate(15rem, 17rem);
}
37.5% {
transform: translate(40rem, 30rem);
}
50% {
transform: translate(0, 0);
}
62.5% {
transform: translate(-10rem, -10rem);
}
75% {
transform: translate(-15rem, -17rem);
}
87.5% {
transform: translate(-40rem, -30rem);
}
100% {
transform: translate(0, 0);
}
}

.wandering-button {
position: absolute;
top: 50%;
left: 50%;
animation: wanderingMotion 30s linear infinite;
}
