Cheat-Sheet for CSS

The most important things explained with an example:

/* This is a CSS comment. It won't affect the styling. */

/* Select the h1 element and make the text red and centered. */
h1 {
color: red; /* Text color is red */
text-align: center; /* Text is centered */
}

/* Select all elements with class "highlight" and make the background yellow. */
.highlight {
background-color: yellow; /* Background is yellow */
}

/* Select all elements within the element with ID "container" and add a border. */
#container p {
border: 1px solid black; /* 1-pixel solid black border */
}

/* Select all elements with class "button" and style them as buttons. */
.button {
background-color: blue; /* Blue background */
color: white; /* White text color */
padding: 10px 20px; /* Padding around the text */
border: none; /* No border */
cursor: pointer; /* Cursor style changes to a pointer on hover */
}

This explains everything you need for a stylesheet

Comments are included using /* ... */.
CSS selectors (e.g., h1, .highlight, #container p) target specific HTML elements based on their tag name, class, or ID.
Properties (e.g., color, background-color) are used to define the styling of the selected elements.
Values (e.g., red, center) are the settings for the CSS properties.
The ; at the end of each property-value pair separates different styles.
The stylesheet can be included in an HTML document using a <link> element in the document's <head> section.
