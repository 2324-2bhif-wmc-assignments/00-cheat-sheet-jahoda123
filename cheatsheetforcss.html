Cheat-Sheet for CSS

CSS Rules

A rule has a selector and a declaration block. Selectors are used to select the element. In the declaration block comes the design changes.

There are 5 different selector types:

  • Simple-selectors
  • Combinator-selectors
  • Pseudo-class selectors
  • Pseudo-elements selectors
  • Attribute-selectors

When you want to change the style of some things to the same you can do this like this:

h2, h3, h4 {
color:black;
}

Comments

Comments in css are started with /* and end with */. They are only visible to the programmer in the css file.

/*This is a css comment.*/

Colors

Color using names

CSS allows specifying colors using predefined names. CSS/HTML support 140 standard color names.

color: red;

Color using HEX-Values

Hexadecimal color is specified with #RRGGBB. RR (red), GG (green), and BB (blue) are hexadecimal values between 00 and ff (equivalent to decimal 0-255). Example: #ff0000 represents red (maximum red, no green or blue). #000000 is black, and #ffffff is white.

color: #ffffff;

Color using 3 digit HEX

A shorthand for some 6-digit hex codes. Takes the form #rgb, where r, g, and b represent red, green, and blue components. The 3-digit code can only be used when RR, GG, and BB values are the same. Example: #ff00cc can be written as #f0c.

color: #f00;

Background

Background Colors

Specifies the background color of an element.
Color can be defined using:
Valid color name (e.g., "red").
HEX value (e.g., "#ff0000").
RGB value (e.g., "rgb(255,0,0)")

background: red;

Background Images

Specifies an image to use as the background of an element. Default behavior repeats the image to cover the entire element. Can be set for specific elements, such as <p>.

background: image.jpg;

Text

Color

The 'color' property sets the color of the text.
Color specified by:
Color name (e.g., "red").
HEX value (e.g., "#ff0000").
RGB value (e.g., "rgb(255,0,0)").
Default text color for a page is defined in the body selector.

/* Setting text color using color name */
body {
color: red;
}

/* Setting text color using HEX value */
h1 {
color: #00ff00; /* Green */
}

/* Setting text color using RGB value */
p {
color: rgb(0, 0, 255); /* Blue */
}

Alignment

'text-align' property sets horizontal alignment.
Options: left, right, center, justify.
'text-align-last' aligns the last line of text.
'direction' and 'unicode-bidi' properties change text direction.
'vertical-align' sets vertical alignment.

/* Aligning text to the center */
h2 {
text-align: center;
}

/* Justifying text */
p {
text-align: justify;
}

/* Aligning the last line to the right */
p {
text-align-last: right;
}

/* Changing text direction to right-to-left */
div {
direction: rtl;
}

Decoration

'text-decoration-line' adds decoration lines to text.
Multiple values can be combined (e.g., overline and underline).
'text-decoration-color' sets line color.
'text-decoration-style' sets line style.
'text-decoration-thickness' sets line thickness.
'text-decoration: none;' removes underline from links.

/* Adding an underline to text */
a {
text-decoration-line: underline;
}

/* Combining overline and underline */
h3 {
text-decoration-line: overline underline;
}

/* Setting the color of the decoration line */
p {
text-decoration-color: #ff4500; /* Orange-Red */
}

/* Removing underline from links */
a {
text-decoration: none;
}

Transformation

'text-transform' specifies uppercase, lowercase, or capitalize.
Options: uppercase, lowercase, capitalize.

/* Transforming text to uppercase */
h4 {
text-transform: uppercase;
}

/* Transforming text to lowercase */
span {
text-transform: lowercase;
}

/* Capitalizing the first letter of each word */
h5 {
text-transform: capitalize;
}

Fonts

Font Family

Use the 'font-family' property in CSS to specify the font of text.
If the font name has more than one word, enclose it in quotation marks (e.g., "Times New Roman").
Include multiple font names as a fallback system for compatibility.

font-family: "Arial", "Helvetica", sans-serif;

Font Style

'font-style' is used for italic text.
Values:
normal: Text is shown normally.
italic: Text is shown in italics.
oblique: Text is "leaning" (similar to italic, but less supported).
'font-weight' specifies font thickness.
'font-variant' determines if text should be displayed in small-caps.

Font Size

'font-size' sets the size of text.
Use proper HTML tags for structure (e.g., <h1> for headings, <p> for paragraphs).
Size can be absolute (specified size) or relative (relative to surrounding elements).

h2 {
font-size: 10px; /* Absolute size in pixels */
}

p {
font-size: 1.5rem; /* Relative size in rem units */
}

Font Weight

'font-weight' sets how thick or thin characters in text should be displayed. Values range from 100 (thin) to 900 (bold).

p { font-weight: 900; } /* or */ p { font-weight: 100; }

Links

a:link -> regular links a:visited -> links that you already clicked on a:hover -> links that you hover with the mouse a:active -> links the moment you click them

/* Normal, Unvisited Link */
a:link {
color: #1e90ff; /* Dodger Blue */
text-decoration: underline; /* Underline the link */
}

/* Visited Link */
a:visited {
color: #800080; /* Purple */
}

/* Link on Mouse Over */
a:hover {
color: #ff4500; /* Orange-Red */
font-weight: bold; /* Bold text on hover */
}

/* Link on Click */
a:active {
color: #008000; /* Green */
}

Combinators

Descendant Selector (Space)

div p {
/* Selects all <p> elements inside a <div> */
color: blue;
}

Matches all elements that are descendants of a specified element.

Child Selector (>)

Selects all elements that are children of a specified element.

ul > li {
/* Selects all <li> elements that are direct children of a <ul> */
list-style-type: square;
}

Adjacent Sibling Selector (+)

Selects an element that is directly after another specific element. Sibling elements must have the same parent, and "adjacent" means "immediately following."

h2 + p {
/* Selects the <p> element immediately following an <h2> */
font-style: italic;
}

General Sibling Selector (~)

Selects all elements that are next siblings of a specified element.

h3 ~ p {
/* Selects all <p> elements that are siblings following an <h3> */
color: green;
}

Pseudo Classes

A pseudo-class is used to define a special state of an element. It can be used for various purposes, such as styling an element when a user interacts with it.

selector:pseudo-class {
property: value;
}

Tables

Black Dotted Border (1px width):

table {
border: 1px dotted black;
}

Solid Border (1px width, Red Color, Rounded Corners - 5px Radius):

table {
border: 1px solid red;
border-radius: 5px;
}

Merge Table Cell Borders (appear as one line):

table {
border-collapse: collapse;
}

Lists

Bullet as a square:

ul {
list-style-type: square;
}

Bullet as an image:

ul {
list-style-image: url("my-bullet.png");
}

Bullet as an uppercase roman number:

ol {
list-style-type: upper-roman;
}

Bullet Position Inside List Item's Text Flow:

li {
list-style-position: inside;
}

Bullet Position Outside List Item's Text Flow:

li {
list-style-position: outside;
}

Box-Model

Block Elements

Block elements take up the full width available and start on a new line. Examples of block elements include:

  • article
  • section
  • p
  • h1 - h6
  • div

div {
width: 300px;
margin: 10px;
padding: 20px;
border: 1px solid #000;
}

Inline Elements

Inline elements, on the other hand, do not start on a new line and only take up as much width as necessary. Examples of inline elements include:

  • img
  • strong
  • em
  • span

/* Example styling for inline elements */
span {
color: red;
text-decoration: underline;
}

Floating Elements

In CSS, floating elements is a technique used to remove elements from the normal flow of the document and position them to the left or right of their containing element. This is often used to create layouts where elements appear side by side.

/* Floating an element to the left */
.float-left {
float: left;
}

/* Floating an element to the right */
.float-right {
float: right;
}

/* Clearing floats */
.clearfix::after {
content: "";
display: table;
clear: both;
}
