Cheat-Sheet for HTML

Basics

Structure

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<link rel="stylesheet" href="./styles.css">
<title>Cheat-Sheet for HTML</title>
</head>
<body>
</body>
</html>

This provides the basic structure for an HTML document. It includes the document type declaration, specifies the language, links an external stylesheet, sets the character encoding, and establishes the basic structure with head and body sections.

Elements

An element starts at the start tag (<start>) and goes till the ent tag (<end>)

Heading Elements

Heading elements range from <h1> (most important) to <h6> (least important). They define the headings and subheadings of a document.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>
<!-- ...and so on -->

Headings are essential for structuring content and are also used by search engines to understand the hierarchy and relevance of information on a web page.

<h1>This is the main heading</h1>
<p>This is some text under the main heading.</p>
<h2>This is a subheading</h2>
<p>More text under the subheading.</p>

Paragraphs

In a paragraph you can write anything you want. This is a paragraph.

<p>This is how you write a paragraph<p>

Links

HTML links are created using the <a> tag. To open a page, use the href attributes with the value "website.at" or with an id you can go to a specific place on the webpage. The text between the tags is what the link will say.

<a href="#">With the id # you go to the top of the page.</a>

Void Elements

Void Elements are empty, like the <br> element, which is used like an enter.

Nested Elements

You can also put elements in different elements. Those are called nested elements.

<div class="outer-container">
<h1>Main Heading</h1>
<p>This is a paragraph within the outer container.</p>
<div class="inner-container">
<h2>Subheading</h2>
<p>This paragraph is nested inside the inner container.</p>
</div>
<p>Another paragraph within the outer container.</p>
</div>

Attributes

HTML attributes provide extra information about HTML elements, influencing their behavior or appearance. They're added to element tags and typically follow a name-value format.

Those are the most common attributes:

  1. src: Specifies the source (e.g., for images or scripts).
  2. href: Defines the hyperlink destination.
  3. alt: Describes alternative text for images.
  4. class: Assigns a class for styling with CSS.
  5. id: Provides a unique identifier for an element.

Comments

HTML comments are used to include notes or remarks within the HTML code that are not displayed in the browser. They are meant for developers and serve as a way to document the code or temporarily disable specific parts without affecting the page's appearance.

<!-- This is an HTML comment -->

Images

Images are very important to improve how your website looks.

For an image you need the tag <img>. There are 2 attributes you have to add. Src and alt. Src is used for the link of the image, alt is for the text which will be displayed if the picture is not available.

SRC Image Link

Absolute Paths look like this:

<img src="/images/image.jpg" alt="Description">

Relative Paths look like this:

<img src="../images/image.jpg" alt="Description">

Links to external Webpages look like this:

<img src="https://example.com/image.jpg" alt="Description">

Head Elements

In HTML, the <head> element contains meta-information about the HTML document, which is not displayed on the web page itself. Here are some common head elements, including the <meta> element.

<title>

Specifies the title of the HTML document, displayed in the browser's title bar or tab.

<title>Page Title</title>

<meta>

Defines the character set for the document. It is commonly set to UTF-8 to support a wide range of characters.

<meta charset="UTF-8">

Configures the viewport settings, ensuring proper rendering on various devices.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Provides a brief description of the HTML document for search engines and social media.

<meta name="description" content="Description of the page">

<link>

Establishes various links, such as linking to stylesheets or external resources.

<link rel="stylesheet" href="styles.css">

<style>

Contains inline CSS styles that apply to the document.

<style> /* Your CSS styles here */ </style>

<script>

Embeds or references JavaScript code.

<script src="script.js"></script>

Tables

Header 1 Header 2

In this example:

<table>: Defines the start of the HTML table.
<thead>: Groups the header content.
<tr>: Represents a table row in the header.
<th>: Defines header cells.
<tbody>: Groups the body content.
<tr>: Represents a table row in the body.
<td>: Defines standard data cells.
<tfoot>: Groups the footer content.
<tr>: Represents a table row in the footer.
<td>: Defines cells in the footer.

This example creates a table with headers, body content, and a footer, demonstrating the use of the specified HTML table elements.
