Cheat-Sheet for HTML

This is an example for html code

<!DOCTYPE html>
<html>
<head>
<title>My HTML Page</title>
</head>
<body>
<h1>Welcome to My Page</h1>
<p>This is a simple HTML page.</p>

<!-- This is a comment. It won't be displayed in the browser. -->

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>

<a href="https://www.example.com">Visit Example.com</a>
</body>
</html>

This explains everything:

In this example:

<!DOCTYPE html> is a declaration specifying the document type and version.
<html> is the root element, containing all the other elements.
<head> contains metadata about the page, like the title in <title>.
<body> contains the visible content of the webpage.
<h1> is a top-level heading.
<p> is a paragraph.
<!-- ... --> is an HTML comment.
<ul> is an unordered list, containing list items <li>.
<a> is an anchor (link) element with an href attribute pointing to a web address.
This example showcases the basic structure of an HTML document with elements like headings, paragraphs, lists, comments, and hyperlinks.
